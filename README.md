This project was created as part of my early programming journey to practice user input, output formatting, and basic arithmetic operations.

➖ Subtraction
✖️ Multiplication

🔼 Exponential powers
📐 Square root calculation
💬 Clean and conversational output

Python (beginner level)

Arithmetic operators
